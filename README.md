
# ✈️ Airport Traffic Data Analysis using SQL

## 📌 Project Overview

This project analyzes airport traffic data using MySQL to uncover meaningful business insights from airline passenger records. The analysis focuses on passenger traffic, flight frequency, airport performance, and route popularity using SQL queries.

The objective is to demonstrate practical SQL skills by solving real-world business problems using aggregate functions, joins, Common Table Expressions (CTEs), and data analysis techniques.

---

## 🛠️ Tools & Technologies

- MySQL
- MySQL Workbench
- SQL
- GitHub

---

## 📂 Dataset

The dataset contains airline passenger traffic information, including:

- Origin Airport
- Destination Airport
- Origin City
- Destination City
- Airline ID
- Passenger Count
- Flight Routes

---

## 🎯 Business Problems Solved

✔ Analyze total passenger traffic by route

✔ Determine average passengers per flight

✔ Identify the busiest flight routes

✔ Measure flight frequency across routes

✔ Evaluate airline seat utilization

✔ Identify the most popular destination airports

✔ Compare passenger traffic with city population
